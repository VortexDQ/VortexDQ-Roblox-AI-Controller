QUICK START (5 MINUTES)
═══════════════════════

1. INSTALL DEPENDENCIES

   Open Command Prompt/PowerShell in this folder and run:

   npm install

   (Takes ~1-2 minutes)

2. START SERVER

   npm start

   Or double-click: start.bat (Windows) / start.sh (Mac)

   You should see:
   🌐 Web UI:  http://127.0.0.1:7777

3. OPEN WEB BROWSER

   Go to: http://127.0.0.1:7777

   You should see a beautiful dark interface.

4. INSTALL ROBLOX PLUGIN

   Copy these 4 files to your Roblox Plugins folder:

   📁 From: plugin/
   ├── Main.lua
   ├── WebSocketClient.lua
   ├── Controller.lua
   └── InstanceManager.lua

   📁 To (Windows): C:\Users\[Your Name]\AppData\Roaming\Roblox\Plugins\
   📁 To (Mac): ~/Library/Application Support/Roblox/Plugins/

   Then restart Roblox Studio.

5. START BUILDING!

   In the web UI, type:
   "create a red part in workspace"

   Click Send and watch Roblox Studio update instantly!

2. Create: ANTHROPIC_API_KEY=sk-ant-...

1. Download: https://ollama.ai
2. Run: ollama serve
3. No key needed!


TROUBLESHOOTING
═══════════════

"Web UI won't load"
  → npm start running? Check terminal
  → Right URL? http://127.0.0.1:7777

"Plugin disconnected"
  → Did you copy 4 .lua files?
  → Did you restart Roblox?
  → Is server still running?

"Commands don't work"
  → Check Errors tab in UI
  → Check instance paths exist
  → Restart server and Roblox

"API key error"
  → Use Ollama instead (free!)
  → Or check key format (sk-ant-...)
